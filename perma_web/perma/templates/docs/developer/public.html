API - Public Archives

The Public Archives API endpoint helps you find all archives in Perma.css that have been Vested and are not in the dark archive. You do not need an API Key to use this potion of the API.

The base endpoint for this API is https://api.{{ request.META.HTTP_HOST }}/v1/public/archives/

Get all public archives

Use an HTTP GET to retrieve a every public archive in Perma.cc. (In order to keep this example short, we limit our number of returned items to one.)

curl https://api.{{ request.META.HTTP_HOST }}/v1/public/archives/?limit=1

Response

{"meta": {"limit": 1, "next": "/v1/public/archives/?limit=1", "offset": 0, "previous": null, "total_count": 2}, "objects": [{"assets": [{"base_storage_path": "2015/1/21/22/39/Y6JJ-TDUJ", "favicon": null, "image_capture": "cap.png", "pdf_capture": null, "warc_capture": "archive.warc.gz"}], "creation_timestamp": "2015-01-21T17:39:44", "dark_archived": true, "dark_archived_robots_txt_blocked": false, "expiration_date": "2017-01-21T17:39:44", "guid": "Y6JJ-TDUJ", "title": "Example.com. yo.", "url": "http://example.com", "vested": true, "vested_timestamp": "2015-01-21T17:49:39", "vesting_org": {"id": 1, "name": "Test Journal"}, "view_count": 1}]}

(Note that "vesting_org" may be null, even if the link is vested, if the vesting org has opted to vest privately.)

Get one archive

If we have a the globally unique ID of one archive, we can GET details on it.

curl https://api.{{ request.META.HTTP_HOST }}/v1/public/archives/Y6JJ-TDUJ/
